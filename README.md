# ansible

1. Install `command line tools`

```bash
xcode-select --install
```

2. Agree to the license

```bash
sudo xcodebuild -license
```

3. Run the bootstrap script

```bash
curl https://raw.githubusercontent.com/UNRULYEON/ansible/main/bootstrap-mac | bash
```
